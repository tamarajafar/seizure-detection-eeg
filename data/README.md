# Data

Raw EEG data is **not stored in this repository**. Download it manually before running any scripts.

## Dataset: CHB-MIT Scalp EEG Database

- **Source:** PhysioNet
- **Citation:** Shoeb A. CHB-MIT Scalp EEG Database. PhysioNet (2010). https://doi.org/10.13026/C2K01R
- **License:** Open Data Commons Attribution License v1.0
- **Size:** ~47 GB (full dataset), 916 hours of continuous EEG

## Download Instructions

### Option 1: wget (recommended)

```bash
wget -r -N -c -np https://physionet.org/files/chbmit/1.0.0/ -P data/raw/
```

### Option 2: PhysioNet web interface

Go to https://physionet.org/content/chbmit/1.0.0/ and download the full dataset.

### Option 3: Subset for development

To download only a few subjects for local development:

```bash
for subj in chb01 chb02 chb03; do
    wget -r -N -c -np https://physionet.org/files/chbmit/1.0.0/${subj}/ -P data/raw/
done
```

## Expected Layout After Download

Using `wget -r` (Options 1 and 3) creates a nested PhysioNet mirror structure:

```
data/
├── raw/
│   └── physionet.org/
│       └── files/
│           └── chbmit/
│               └── 1.0.0/
│                   ├── chb01/
│                   │   ├── chb01_01.edf
│                   │   ├── chb01_02.edf
│                   │   └── chb01-summary.txt   # seizure annotations
│                   ├── chb02/
│                   │   └── ...
│                   └── ...  (chb01 through chb24, skipping chb04)
└── processed/                   # generated by preprocessing scripts
    ├── segments_X.npy
    ├── segments_y.npy
    └── subject_ids.npy
```

The preprocessing pipeline automatically detects both this nested structure and a flat layout (`data/raw/chbXX/`).

## Subject Notes

- 23 subjects total (chb01-chb24; chb04 is missing from the dataset)
- Ages 1.5-22 years, pediatric patients with intractable focal epilepsy
- 198 annotated seizures across all subjects
- Recordings collected at Children's Hospital Boston after medication withdrawal
- 23 EEG channels, international 10-20 system, sampled at 256 Hz
